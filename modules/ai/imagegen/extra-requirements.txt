# Baked pip deps for custom-node packs. Workflow: clone a node into custom_nodes/ → add its
# requirements.txt entries here → `docker compose build` → restart. Never rely on runtime
# `pip install` (it dies with the container).
#
# NEVER add torch / torchvision / torchaudio here — the ROCm base image owns those. A node that
# pins them gets neutralized the same way the Dockerfile grep-filters ComfyUI's own requirements.
#
